# 🏫 Scalable Campus Area Network using Cisco Packet Tracer

## 📖 Overview

This project demonstrates the design and implementation of a **Scalable Campus Area Network (CAN)** using **Cisco Packet Tracer**. The network is designed to provide secure, reliable, and efficient communication between multiple departments within a campus environment. It follows a modular architecture, making it easy to expand as the organization grows.

---

## 🎯 Objectives

- Design a scalable campus network architecture.
- Implement efficient IP addressing and subnetting.
- Configure VLANs for network segmentation.
- Enable communication between different VLANs.
- Improve network performance and security.
- Simulate a real-world enterprise campus network.

## 🌐 Network Architecture

The campus network follows a hierarchical design consisting of multiple switches connected to routers. Different departments are placed in separate VLANs, allowing secure communication while maintaining efficient traffic management.

Example Departments:

- Administration
- Computer Science
- Library
- Faculty
- Students
- Accounts
